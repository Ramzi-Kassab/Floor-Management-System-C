File Format Requirements

Required Columns (in order):

  1. item_code - Unique item code
  2. name - Item name
  3. category_code - Category code (must exist)
  4. uom_code - Unit of measure code (must exist)
  5. item_type - Type (CUTTER, MATRIX_POWDER, etc.)
  6. reorder_level - Reorder quantity
  7. description - Item description (optional)
  8. supplier_code - Supplier code (optional)

Valid Item Types:

  • CUTTER
  • MATRIX_POWDER
  • STEEL_BODY
  • JV_PARTIAL_BODY
  • UPPER_SECTION
  • HARDFACING
  • CONSUMABLE
  • TOOL
  • SPARE_PART
  • OTHER
Note: First row should contain column headers. If an item with the same code exists, it will be updated.
