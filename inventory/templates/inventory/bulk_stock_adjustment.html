File Format Requirements

Required Columns (in order):

  1. item_code - Item code (must exist)
  2. location_code - Location code (must exist)
  3. condition_code - Condition type code (must exist)
  4. ownership_code - Ownership type code (must exist)
  5. quantity - Adjustment quantity (can be negative)
  6. reference - Reference number (optional)
  7. notes - Adjustment notes (optional)
About Quantities:
  • Positive numbers = Add stock
  • Negative numbers = Remove stock
Note: First row should contain column headers. All referenced items, locations, conditions, and ownership types must exist in the system.
